$$
题001\\

已知{\kern 2pt}a+b=1,(a>0,b>0),求{\kern 2pt}\frac{1}{a^2}+\frac{8}{b^2}的最小值\\

解:\frac{1}{a^2}+\frac{8}{b^2}=\frac{1^3}{a^2}+\frac{2^3}{b^2}\geq\frac{(1+2)^3}{(a+b)^2}=3^3=27
$$

$$
题002\\

已知{\kern 2pt}\frac{8}{a}+\frac{1}{b}=1,(a>0,b>0),求{\kern 2pt}a^2+b^2的最小值\\

解:令{\kern 2pt}m=\frac{8}{a},n=\frac{1}{b}\\

原式=\frac{64}{m^2}+\frac{1}{n^2}=\frac{4^3}{m^2}+\frac{1^3}{n^2}\geq\frac{(4+1)^3}{(m+n)^2}=5^3=125
$$
